# BluCoop - Cooperativa

`CURSO: Sistemas de Informação`

`DISCIPLINA: Projeto - Projeto de Infraestrutura`

`Eixo: 5`

O projeto propõe a implantação de uma infraestrutura de rede para a cooperativa BluCoop, conectando a matriz e suas unidades por meio de uma comunicação segura, confiável e escalável. A solução também inclui uma aplicação web em nuvem para apoiar o gerenciamento das unidades, integrando banco de dados e práticas de segurança para proteger as informações e sustentar as operações da organização.
