# Cisco Packet Tracer

Esta pasta contém o protótipo da infraestrutura de rede da BluCoop desenvolvido no Cisco Packet Tracer para a Etapa 1.

## Arquivo

| Arquivo | Descrição |
| --- | --- |
| [blucoop_projeto.pkt.zip](./blucoop_projeto.pkt.zip) | Pacote compactado contendo o arquivo `.pkt` do protótipo de rede. |

## Conteúdo do Pacote

O arquivo `blucoop_projeto.pkt.zip` contém o projeto:

```text
blucoop_projeto.pkt
```

Esse arquivo deve ser aberto no Cisco Packet Tracer.

## Como Abrir

1. Baixe ou extraia o arquivo `blucoop_projeto.pkt.zip`.
2. Descompacte o pacote para obter `blucoop_projeto.pkt`.
3. Abra o arquivo `.pkt` no Cisco Packet Tracer.

Exemplo pelo terminal:

```bash
unzip blucoop_projeto.pkt.zip
```

O protótipo representa a proposta de rede da cooperativa, incluindo matriz, filiais, segmentação, equipamentos, endereçamento e testes de comunicação. Ele complementa a documentação principal da Etapa 1:
