# Projeto BluCoop

Repositório do projeto de infraestrutura e segurança da informação desenvolvido para a BluCoop, uma cooperativa de crédito regional. O trabalho reúne planejamento de rede, implantação de serviços locais e em nuvem, monitoramento, aplicação web, política de segurança, evidências de entrega e materiais finais de apresentação.

## Visão Geral

O projeto foi estruturado em cinco etapas:

| Etapa | Foco |
| --- | --- |
| Etapa 1 | Planejamento da infraestrutura de rede, topologia, endereçamento IP e protótipo no Cisco Packet Tracer. |
| Etapa 2 | Implantação de serviços locais e em nuvem, incluindo AD, DHCP, DNS, PostgreSQL, FTP, EFS, CloudFront e servidor web. |
| Etapa 3 | Monitoramento dos serviços com Zabbix, dashboards, gráficos, hosts e mapas de topologia. |
| Etapa 4 | Desenvolvimento web backend, Política de Segurança da Informação, folder de conscientização e análise OWASP. |
| Etapa 5 | Conclusão, referências, apresentação final e fechamento do projeto. |

## Principais Entregáveis

- Documentação final do projeto em PDF.
- Entregas por etapa em Markdown.
- Protótipo de rede no Cisco Packet Tracer.
- Código do servidor web com frontend, backend Node.js/Express e configuração Apache.
- Documentação da EC2 usada como servidor web.
- Política de Segurança da Informação e folder institucional.
- Ata de acompanhamento do projeto.
- Vídeos demonstrativos das etapas.
